# FFXI-Windower-AbysseaFarmTracker
Keep track of needed pop items and more when farming items in Abyssea

This is work in progress, but the idea behind it is to make it easier to keep track of what items you currently have and or need to spawn/pop enemies for things like Glavoid Shell.
